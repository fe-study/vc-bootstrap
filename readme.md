## vc-bootstrap

### vc components

* vcTabs
* vcPagination
* vcModal
* vcBacktop
* vcTable
* vcEacyclearinput
* vcAutocomplete
* vcEchart
* vcSelect

### 如何参与

1. 安装 iat 命令行工具一键创建组件框架
2. 开发单个组件，每个组件分别对应一个单独项目，并以npm包的形式单独发布
3. 合并到主仓库 vc-bootstrap 中

## usage

* Build

```node
npm run build
```

* Build example

```node
npm run example
```

* run server

```node
npm run start
```
